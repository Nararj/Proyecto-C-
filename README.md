# Proyecto-C-

- Programa que permita manejar distintas plantas, caracteristicas y cuidados.
- Permite crear diferentes plantas y almacenarlas en un arreglo, mostrar información de todas las plantas, regarlas, cosecharlas o podarlas segun tipo de planta.

**ALGORITMO**

*ENTRADAS*
- Tipo de planta, nombre de la planta, altura, cantidad de agua diaria

*ALGORTIMO*

1.- Inicializar vector/lista para guardar plantas.
2.- Mostrar menú principal:
  2.1.- Agregar nueva planta
  2.2.- Mostrar información de todas las plantas
  2.3.- Regar todas las plantas
  2.4.- Realizar acción específica por tipo
  2.5.- Salir
3.- Leer opción del usuario.

*SALIDAS*
- Confirmación de creación de planta
- Información de cada planta
